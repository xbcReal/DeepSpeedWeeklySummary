🚀 一、合入的 PR — 重要特性
Feature #8066 Mixed-precision: per-policy param/buffer dtype cast (preserve fp32 buffers)
作者: @sfc-gh-truwase | 合入: 2026-06-16
新增 data_types.param_dtype 和 data_types.buffer_dtype 配置（均默认 None），对标 FSDP 的 MixedPrecisionPolicy。将原来的 module.half() / module.bfloat16() 全局转换替换为更精确的按策略转换：参数按 param_dtype 转换，浮点 buffer 按 buffer_dtype 转换。这对于需要保留 fp32 buffer 的场景（如 BatchNorm 的 running_mean/var）非常重要。
Feature #8054 Add Biren SUPA accelerator support
作者: @frozenleaves | 合入: 2026-06-16
为壁仞科技 SUPA GPU 新增加速器后端支持。DeepSpeed 现在可以自动检测壁仞设备，在壁仞 GPU 上运行训练和推理。通过复用 DeepSpeed 已有的 CUDA-compatible 加速器抽象框架实现，进一步扩展了 DeepSpeed 的硬件生态覆盖。
