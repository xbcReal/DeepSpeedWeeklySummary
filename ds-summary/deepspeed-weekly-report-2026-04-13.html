Major Features

Add AutoEP (Automatic Expert Parallelism)
#7938
Review tohtana Mar 31
为 HuggingFace MoE 模型添加 AutoEP 自动专家并行功能。在 deepspeed.initialize() 期间检测 MoE 块，构建 EP/EDP 进程组，将支持的 MoE 块替换为 EP 启用执行路径。支持 Mixtral、Qwen3-MoE、DeepSeek-V2/V3、LLaMA-4 等模型。当前支持 ZeRO stages 0/1/2，包含 checkpoint 和 universal checkpoint 支持。
Enable AutoEP Compatibility with ZeRO-3
#7928
Review nathon-lee Mar 28
实现 AutoEP 与 ZeRO-3 的兼容性。通过选择性绕过 ZeRO-3 的专家参数机制，同时保持非专家参数的完整 ZeRO-3 分片，解决了专家并行参数分区与 ZeRO-3 数据并行分片之间的冲突。
Add Gram Newton-Schulz orthogonalization for Muon optimizer
#7953
Review delock Apr 3
为 Muon 优化器集成 Gram Newton-Schulz 正交化方法。相比标准 Newton-Schulz，Gram NS 在更小的 Gram 矩阵上迭代，在 transformer 权重矩阵常见的高宽比场景下显著降低计算成本。添加 ns_method 参数支持在 gram 和 standard 之间切换。
Add torch_xla TPU support for ZeRO-1/2
#7917
Review PKUWZP Mar 21
添加 XLA 加速器以通过 torch_xla 在 TPU 上执行。使用 torch.distributed xla 后端和 xla:// 初始化流程初始化 DeepSpeed 分布式。使 ZeRO-1/2 梯度分桶后端无关化，并添加 TPU/XLA 专用测试和文档。
Add CPU offload support for Muon optimizer in ZeRO2
#7939
Review delock Mar 31
在 ZeRO Stage 1&2 CPU 卸载路径中添加 Muon 优化器支持。动量存储在 CPU 内存中，Newton-Schulz 算法在 GPU 上执行。使 CPU 卸载的 ZeRO2 与非卸载版本具有相同的数值行为。
